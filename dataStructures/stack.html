Stack.

A stack of books is a great example of the stack data structure.

If you make a stack of books, the top-most one was the Book you placed on the stack last. If you remove it from the stacks top, you expose the item that was placed in the stack just before it.

Stacks are LIFO organized (Last in, First Out) type of service

Good example:

Browser's back button : history of previously visited sites is place into a stack and you traverse in a backwards order when hitting the back button

Typical JS functions associated with Stacks

  • push() - add data to stack
  • pop() - remove from stack
  • peek() - displaying top element of a stack
  • length() - how many elements are on the stack

The above methods are native to accessing data from an Array, so in many ways and array can work/act like a stack.

Code example: Palindrome

let letters = []

const word = "racecar"

let testPalindrome = ""

Put letters of word into stack

for(let i = 0; i < word.length; i++ ){ letters.push(word[i]) }

Pop off the stack in reverse order

for(let i = 0; i < word.length; i++ ){ testPalindrome += letters.pop() } if(testPalindrome === word) {console.log(`$:{word} is a palindrome`)}
